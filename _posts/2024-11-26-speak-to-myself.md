# Langauge Learning Strategy: Speaking to Myself

## Introduction

In my previous posts about language learning, I discussed various methods I've been trying to improve language skills. As I have been doing trials and errors by myself, I've tried a few methods that I thought would be effective. Today, I want to focus on a specific technique that I've been experimenting with recently: speaking to myself in the target language (Now I'm working this method on English). As I've been doing this method for a while, I want to share my recent observations and thoughts on this method.

## Why Talk to Yourself?

As my language skill improves, I found that I can understand most of the things I listen to or read in the target language. However, these are mainly passive understanding, where I think I'm still somewhat struggling with the active outputting of the language. I believe passive understanding is important and is a great foundation for active outputting (I even think listening a lot would help with the speaking skill as well). However, at some point, I found that I need to actively practice the outputting to improve my speaking skill.

One of the main reasons why I decided to try this method is that I don't have a partner to practice with. I live in a country where English is not the native language, so it's not easy to find a language exchange partner. In addition, I don't have much time to spare for practice language even if I find a partner (I want to focus on my work and research). Thus, I found speaking to myself is one way to solve this problem. And even it might be more effective than finding a language exchange partner 🤔.

## How Talking to Myself Works

The method is quite simple. I just talk to myself in the target language. I don't care about the content, but just talk to myself like I'm talking to another person or I'm just talking to myself about what I'm thinking. The main point of this excercise is that there are always a point where I get stuck. This means that, if it was a real conversation, I would ask someone else for help. However, in this case, I need to solve it by myself, where the learning comes from the process of reinforcing the natural expressions. When I get stuck, usually there are two ways to solve it: 1. think about the alternative expressions I can use, 2. search the web for the best expression. For the first one, It is a good way to reinforce my process of thinking towards more natural and accurate expressions. For example, think of the situation where I want to say "I want to go to the store". In this case, I might want to mention that I "really" want to go to the store, or I just have to go to the store by my mom's order. So I can think about the alternative expressions if I'm stuck with what I "really" want to say, and reinforce myself to use the most probable one such as "I need to go to the store" or "I have to go to the store". This applies to other contexts as well. By reinfocing my brain to think more probable expressions, I can gradually improve my outputting skill towards more natural expressions. For the second one, there might be a case where I cannot think of the best expression at the moment. In this case, I can search the web for the most probable one. By doing this, I can learn new expressions that I might not know before, and I can use them later when I remember the context.

Some important points however are:
1. Self-talk (from what I am mentioning) is different from shadowing. Shadowing is more about imitating the speaker's intonation and rhythm, where I can improve my accent and pronunciation. However, self-talk is more about expressing my thoughts and ideas, where I can improve my outputting skill in a more creative way.
2. For this strategy to work, I need to absorb a lot of inputs in the target language. This is because, to be able to reinforce my process of creating expressions on the given context, I need to have a strong foundation knowledge to draw from. And I also think that the more you get more inputs, the more you can pick the best expression from the context. Also recently acquired expressions are more likely to be picked when I need to create an expression on the fly.
3. I'm also writing journals for my outputting skill. However, the point of journals is more about writing a structured content, where it is slightly different from conversational self-talk.

### Short description of my current method

In more detail, I want to elaborate a little bit on my current routine. Beforehand, I tried to talk to myself in the target language when I am in the way to work. However, I found that doing the practice in the repetative environment is not that effective because I couldn't think of a variety of topics in the same environment. So the first important thing is to make a list of topics ahead of time. And followings are the details of my current routine.


1. Make a list of topics that I want to talk about the day before.
    - Topics can be about anything (e.g., daily life, work, research, etc.)
    - It can also be describing situation, emotion, thought, object, and so on.
2. Make a time slot where you can extensively practice the topic.
    - e.g., 10 minutes before work, 30 minutes before dinner, etc.
    - For me, I usually do this on the way to work in the morning, where I have around 15-30 minutes.
3. Whenever I get stuck, I should do either of the following: 1) Think about the alternative expressions I can use. 2) Search from the web to fine the best one.
    - This is where I can learn new expressions and improve my outputting skill. Either I reinforce my brain towards more natural expressions or I can learn new expressions to use later.

## Conclusion

Still, this is an on-going project for me. So I need to keep doing this and see how it goes. But so far, I think it's a good method to practice outputting skill. And that is why I want to share this with you guys. I hope this would be helpful for you who are also struggling with the outputting skill.
