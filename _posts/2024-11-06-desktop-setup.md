## Introduction

As a computer science major as well as a graduate degree in AI, I've always dreamed of building my own computer. Building my own workspace, setting up my personal server, and doing various experiments and projects have always been my dreams for a long time.

However, various circumstances kept postponing this project. Living in a cramped dormitory and later a small studio apartment made it impractical to set up a desktop workspace. Additionally, with my hometown being in Jeju island, Korea, and my university and graduate school located on the mainland, purchasing and assembling a desktop at home wasn't feasible either.

But now, I finally decided to build my own computer, not because I'm now living in a big house with a big room, but because I cannot no longer postpone this project (my desire to build my own computer is stronger than my current living conditions 😂). That's why I'm writing this post to document my journey to building my own computer. In addition to having fun assembling a machine from scratch, I strongly think creating a personal workspace would serve multiple purposes. One of my main goals was to configure the desktop as a personal server, opening up possibilities for various technical experiments and projects. As an AI student, having dedicated GPU power would enable me to dive deeper into machine learning projects that I've been eager to explore. And of course, I'm super excited about using this computer for fun stuff too (gaming, watching movies though I don't have much time for that 😅).

In the following sections, I'll take you through my carefully chosen desktop configuration, explaining how each component aligns with my workflow and aspirations. This build represents not just a collection of hardware, but the realization of a long-held dream and the foundation for my future projects.

## Process

### 1. Choosing Components
The first thing I need to do is to choose components for the computer. And It should be considered that I need to choose components that are compatible with each other. Without difficulty, I chose the components for the computer. And because my purpose is to build a personal server as well as a machine to do various experiments and projects, I chose the components that are powerful enough to support my goals.

Note that I'm not a professional computer builder, so I might not be able to explain the process in a professional way. But I'll do my best to explain the process in a way that is easy to understand.

### 2. Install CPU to the mainboard

<div style="text-align: center">

After choosing the components, I started to install the CPU to the mainboard. The process was quite easy. I just need to remove the cover of the mainboard, and then put the CPU in the right place. But in this process, It should be considered that I need to be careful not to damage the CPU or the mainboard.

### 3. Install RAM

<div style="text-align: center">
<img src="/assets/images/posts/desktop-setup/memory.jpg" width="70%" alt="Installing RAM">
</div>

<div style="text-align: center">
<em>Installing RAM to the mainboard</em>

The next thing I need to do is to install the RAM to the mainboard. This process, as well, is not difficult. The thing I need to do is just to put the RAM in the right slots. But one thing that should be considered is that, when I put the RAM in the slots, I need to put the RAM in the right way. That is, most motherboards have 4 RAM slots, and for optimal dual-channel performance, you typically want to install the RAM sticks in the first and third slots (or according to your motherboard's manual).

Now I have to install the M.2 SSD to the mainboard. To install the M.2 SSD, I need to remove the cover of the mainboard. Opps.. Here I realized that my screw driver is not fit into the screw hole of the mainboard. This makes me wait another day to continue the process.

The next day, I finally got the right screw driver and installed the M.2 SSD. And then I successfully installed M.2 SSD to the mainboard. Now I can move on to the next step, installing the mainboard to the case.

### 5. Install mainboard to the case

<div style="text-align: center">

</div>

<div style="text-align: center">

Now I need to move on to installing mainboard to the case. The first thing I need to do is to open the case, and then it seemed easy at first glance. Things went smoothly at first. I put the mainboard in the right place, and then I started to screw the mainboard to the case. And then opps.. I forgot to put additional standoffs for the mainboard. This makes me take out the mainboard from the case, and then I put the additional standoffs. And then I put the mainboard back to the case, and then I started to screw the mainboard to the case again.

It was until now that I realized that I forgot to install the last standoff for the mainboard. And then, holy smokes.. The standoff was ran out of stock. I was able to put 8 out of 9 standoffs, and then decided to be satisfied with the result.

### 6. Install Cooling System

<div style="text-align: center">
<img src="/assets/images/posts/desktop-setup/cooling_0.jpg" width="70%" alt="Installing Cooling System">
</div>

<div style="text-align: center">

There are a variety of cooling systems for the CPU. That being said, I need to carefully read the instruction manual for the cooling system I purchased. Oh, It seems quite easy to install the cooling system. I just need to attach holders to the mainboard, and then apply thermal paste to the CPU, and then attach the cooling system to the CPU.

But the thing has happened as usual. I could not remove the screw from the mainboard, due to the little driver I bought yesterday. I should have bought a bigger driver, where I could remove the screw from the mainboard with enough force. Here, I had to wait another day to buy a bigger driver.

After waiting another couple of days, I finally got the bigger driver, and then I installed the cooling system to the mainboard. To be able to install the cooling system, first thing I need to do is to remove the screw from the mainboard around the CPU. After removing the screws, I attached the mounting brackets that came with the cooling system to the mainboard. Then, I carefully applied thermal paste to the CPU and mounted the cooling system on top with care.

The next thing I need to do is to install the GPU. Installing the GPU is quite easy. I just need to attach the GPU to the mainboard with the help of the bracket that came with the GPU. And the reason why I bought 16GB of VRAM is that I want to have more VRAM for my deep learning projects 🔥.

Now we are ready to install the PSU. PSU is the power supply unit, and it is the most important component in the computer, because it provides power to the computer. Here, I put the PSU on the back bottom of the case, allowing enough space for the PSU's ventilation and also making it easier to connect cables. And I'm now ready to connect cables.

This is by far the most difficult part of the process. I need to connect cables to each component. First thing that I need to make sure is that I should manage cables properly, tie cables together, and then connect cables to each component, in order to make the computer look organized as well as to make it easier to manage cables. Then I connected cables from the PSU to the mainboard, and then connected cables from the mainboard to each component. And then I also connected cabled from the case to the mainboard as well.

Here, I made two mistake. The first one is that I connected cable for booting the computer to the port for rebooting the computer. This caused the computer to reboot when I turned on the computer. And the second one is that I forgot to connect the cable for the case fans, which is easy to notice the problem when I turned on the computer. Anyway all problems were solved after I carefully checked all cables and then connected cables again.

Finally, I booted up the computer, and then I was able to enjoy the computer. And these are the remaining processes that I need to do:

First, I need to install the operating system (You'll need a bootable USB drive with your chosen OS. I highly recommend creating this beforehand). For me, I already had a bootable USB drive with a variety of operating systems such as Windows, Ubuntu, and CentOS, so I didn't need to create a bootable USB drive again. After that, I need to install the drivers for the components - I need to install drivers like the GPU driver to use the components properly. Then, I need to install the necessary software depending on the purpose of the computer. Finally, I can enjoy using the computer!

This is the end of my journey to building my own computer. I'm now really excited to use this computer for various experiments and projects!

Hasta la vista!
